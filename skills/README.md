# Skills

Claude Code skills are markdown prompt files that define custom slash commands (e.g. `/my-skill`).

## Installing a skill

1. Copy the `.md` file for the skill you want into your `~/.claude/skills/` directory (or the project-level `.claude/skills/` directory).
2. Restart Claude Code — the skill will appear as a slash command.

## Creating a new skill

Copy `_template.md`, rename it, and follow the instructions inside. See [`docs/skill-authoring.md`](../docs/skill-authoring.md) for full guidance.
